RF analysis example¶

In this notebook we will explore how we can use this package to infer the properties of a center-surround receptive field model from some synthetic data.

Define the data¶

First of all, we need to generate some synthetic dataset for our analysis. We will generate a noise stimulus $\mathbf{S}$ and a vector of spike counts $\mathbf{c}$. We will model the spike counts as completely independent form the stimulus: the inference should recover the prior distributions in this case, except for the average firing rate and the stimulus coupling parameters.

Receptive Field model¶

We will use a space-time, center-surround receptive-field model. The model consists of a spatial kernel $\mathbf{k}_s$ and of a temporal kernel $\mathbf{k}_t$.

Space RF¶

The space component $\mathbf{k}_s$ is modelled using a parametric kernel, such that $[\mathbf{k}_s]_{ij} = f(x_i, y_j)$, where:

$$\begin{equation} f(x, y) = \frac{1 + \alpha}{2}\frac{1}{\pi\sigma_{\mathrm{pos}}^2}\exp\bigg(-\frac{x^2 + y^2}{2\sigma_{\mathrm{pos}}^2}\bigg) - \frac{1 - \alpha}{2}\frac{1}{2\pi\sigma_{\mathrm{neg}}^2}\exp\bigg(-\frac{x^2 + y^2}{2\sigma_{\mathrm{neg}}^2}\bigg) \end{equation},$$

where $\sigma_{\mathrm{pos}}$ and $\sigma_{\mathrm{neg}}$ control the size of the positive and of the negative region of the receptive-field, and $\alpha \in [-1, 1]$ weights the contribution of the positive and of the negative regions.

We will assume uniform priors $\mathrm{U}(1, \sigma_{\mathrm{max}})$ for the scale parameters $\sigma_{\mathrm{pos}}$ and $\sigma_{\mathrm{neg}}$ and a uniform prior $\mathrm{U}(-1, 1)$ for $\alpha$.

As MCMC works best with unbounded variables, we will work with variables in a transformed space:

  1. $\alpha = \tanh(\alpha')$: $\alpha \sim \mathrm{Uniform}(-1, 1) \rightarrow \alpha' \sim \mathrm{Logistic}(0, \frac{1}{2})$

  2. $\sigma_x = 1 + (\sigma_{\mathrm{max}} - 1) \frac{1}{1 + \exp(-\sigma_x')}$: $\sigma_x' \sim \mathrm{Logistic}(0, 1)$

from maprf import shapes
from maprf import transform

SIGMA_MAX = kx // 3

# create a kernel and add its parameters to the model
with model:
    _shape = addparam('shape', scalar(), prior=dist.Logistic(0., .5))
    _w_pos = addparam('w_pos', scalar(), prior=dist.Logistic(0., 1.))
    _w_neg = addparam('w_neg', scalar(), prior=dist.Logistic(0., 1.))
    
    ks = shapes.CenterSurround(
        shape=transform.arctanh.inverse(_shape),
        w_pos=transform.Interval(1., SIGMA_MAX).inverse(_w_pos),
        w_neg=transform.Interval(1., SIGMA_MAX).inverse(_w_neg))
    
    # add two parameters for the rf position
    xo = addparam('xo', scalar(int))
    yo = addparam('yo', scalar(int))
    
# add a stage to the model filter
with model.filter:
    rfs = filters.SpatialFilter(ks[grid])

Defining the rate model¶

As we saw, the receptive field consists of a temporal and a spatial linear component. We must then ensure that the output of the linear stage is rectified in order to represent a proper firing rate. The output of the linear stage is rectified by a static non-linearity, in this case we will use $\exp$. Before going throught the non-linearity, we will rescale and shift the output of the linear stage in order to accomodate the baseline firing rate and the coupling strenght between stimulus and firing rate of the neuron.

We have the following model for the static non-linearity (assuming that $z_t$ is the output of the linear stage at time $t$):

$$ r_t = \exp(a + \mathbf{b}^{\top} \mathbf{z}_t) $$

In this particular case, $\mathbf{z}_t$ is a scalar variable, and it's amplitude was already shaped by the temporal component of the receptive field. Therefore, we set $\mathbf{b} = [1]$.

Sampling¶

To make sampling efficient, we group the model parameters in two categories: the receptive field position $\mathbf{x} = \{x_o, y_o\}$ and the parameters controlling the receptive-field appearance $\boldsymbol{\theta}$.

We want to sample from the posterior distyribution $p(\mathbf{x}, \boldsymbol{\theta}|\mathcal{D})$, which is proportional to the joint distribution of data $\mathcal{D}$ and parameters $p(\mathcal{D}, \mathbf{x}, \boldsymbol{\theta}) = p(\mathcal{D} | \mathbf{x}, \boldsymbol{\theta})p(\mathbf{x})p(\boldsymbol{\theta})$. The posterior can be rewritten as $p(\mathbf{x}, \boldsymbol{\theta}|\mathcal{D}) = p(\mathbf{x}| \boldsymbol{\theta},\mathcal{D}) p(\boldsymbol{\theta}|\mathcal{D})$, where $p(\boldsymbol{\theta}|\mathcal{D})$ is the posterior distribution of the receptive-field appearance independently of the receptive-field position.

$p(\boldsymbol{\theta}|\mathcal{D}) \propto p(\boldsymbol{\theta}, \mathcal{D})$ which, for discrete $\mathbf{x}$, is

$$ p(\boldsymbol{\theta}, \mathcal{D}) = \sum_i p(\mathcal{D}, \mathbf{x}_i, \boldsymbol{\theta}) = \sum_i p(\mathcal{D} | \mathbf{x}_i, \boldsymbol{\theta})p(\mathbf{x}_i)p(\boldsymbol{\theta}) = p(\boldsymbol{\theta}) \bigg(\sum_i p(\mathcal{D} | \mathbf{x}_i, \boldsymbol{\theta})p(\mathbf{x}_i)\bigg) $$

The conditional probability $p(\mathbf{x}| \boldsymbol{\theta},\mathcal{D})$ can be written as:

$$ p(\mathbf{x}| \boldsymbol{\theta},\mathcal{D}) = \frac{p(\mathcal{D} | \mathbf{x}, \boldsymbol{\theta})p(\mathbf{x})p(\boldsymbol{\theta})}{p(\boldsymbol{\theta}, \mathcal{D})} = \frac{p(\mathcal{D} | \mathbf{x}, \boldsymbol{\theta})}{\sum_i p(\mathcal{D} | \mathbf{x}_i, \boldsymbol{\theta})p(\mathbf{x}_i)} $$

At this point, we have a simple procedure to sample a pair $\big(\mathbf{x}^{(k)}, \boldsymbol{\theta}^{(k)}\big)$ from their posterior distribution w.r.t. the data:

  1. sample $\boldsymbol{\theta}^{(k)} \sim p(\boldsymbol{\theta} | \mathcal{D})$

  2. sample $\mathbf{x}^{(k)} \sim p(\mathbf{x}| \boldsymbol{\theta}^{(k)},\mathcal{D})$

  3. repeat from 1. until enough samples have been collected

with model.define_sampler():
    s = theano.shared(features, 'X')
    r = theano.shared(response, 'Y')
    
    log_pr = model.log_pr()
    loglik = model.loglik([s, r])
    # posterior over parameters, marginalized (xo, yo)
    target1 = utils.logsumexp(loglik) + log_pr
    # conditional probability of (xo, yo) given the other parameters
    target2 = loglik - utils.logsumexp(loglik)
    
    # define the steps of the sampler
    step1 = addstep('hmc', target1, [_shape, _w_pos, _w_neg, a], stepsize=1e-3)
    step2 = addstep('pos', target2, [xo, yo])
    model.sampler.setup()
